Beego Framework
一个使用 Go 的思维来帮助您构建并开发 Go 应用程序的开源框架

Quick start

package main

import "github.com/astaxie/beego"

func main() {
    beego.Run()
}
go build hello.go
./hello

Features

  • RESTFul support
  • MVC architecture
  • Session support (store in memory, file, Redis or MySQL)
  • Cache support (store in memory, Redis or Memcache)
  • Global Config
  • Intelligent routing
  • Thread-safe map
  • Friendly displaying of errors
  • Useful template functions
